Usage: arc <destination> <source> [options]

Параметры:

Имя архива (первый аргумент - destination)

✔ Абсолютный - т.е. в имени присутствует полный путь к файлу архива, например "D:\Data\archiv". Расширение указывать необязательно, оно проставится само.

✔ Относительный, это означает, что путь будет строится относительно текущего каталога, например ".\Data\archiv". Расширение так-же проставится автоматически.

✔ По умолчанию, это когда указываешь толко само имя архива. Располагаться он будет на диске D:\ в корне. Например такой командой: arc new-archive

Данные для архивации (второй аргумент - source)

✔ Текущий каталог. В этом случае можно вообще его опустить, не указывать. В таком случае заархивируется вся текущая каталог, вместе с подпапками.

✔ Абсолютный путь к данным: если Вы хорошо представляете где расположены Ваши данные - уазываете это место. Например - E:\Data\Archives. В итоге в архиве будет расположен каталог Archives
